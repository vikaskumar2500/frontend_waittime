# 🚀 GovWait – Frontend (Public Wait Time Analytics)


## 📌 Project Description

GovWait is a frontend application designed to provide real-time visibility of waiting times in public offices.

The system improves transparency and citizen experience by displaying:
- Estimated wait times  
- Crowd levels  
- Basic analytics  

This project focuses on building a scalable and modular frontend for data-driven governance.

---

## ✨ Features

- 📍 View nearby offices and wait times  
- ⏱ Real-time wait time display (mock/API-based)  
- 📝 User check-in interface  
- 📊 Admin dashboard (analytics overview)  
- 🔄 Feature-based modular architecture  
- ⚡ Scalable and maintainable structure  

---

## 🧠 Tech Stack

- **Framework:** Next.js (App Router)  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS + shadcn UI  
- **State Management:** Zustand  
- **Forms:** React Hook Form  
- **Validation:** Zod  
- **API Handling:** Axios

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/vikaskumar2500/frontend_waittime.git
cd frontend_waittime
```

---

### 2. Install Dependencies

```bash
pnpm install
```
---

### 3. Run the Project

```bash
pnpm dev
```

Open in browser:

```
http://localhost:3000
```

---

## 📁 Project Structure

```
src/
├── app/                # Routing (Next.js App Router)
├── components/         # UI and reusable components
├── features/           # Feature-based modules (auth, wait-time, analytics)
├── store/              # Zustand global state
├── lib/                # API client, utilities, validators
├── hooks/              # Custom hooks
├── mock/               # Mock data
├── types/              # TypeScript types
```
